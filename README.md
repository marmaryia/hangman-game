# Hangman

With a positive theme.

## Stack

React, Typescript, CSS \
Icons: [Freepik](https://www.freepik.com/) \
Header background: [Vecteezy](https://www.vecteezy.com/) \
Dictionary: [English Dictionary](https://github.com/adambom/dictionary)

## Features

- Single-player mode: guess a word randomly chosen from the dictionary
- Two-player mode: guess a word typed in by another player
- On-screen keyboard dynamically showing correctly/incorrectly guessed letters
- Word definition displayed at the end of the game

## To play

- Clone the repository, navigate into the directory
- `npm i`
- `npm run dev`
